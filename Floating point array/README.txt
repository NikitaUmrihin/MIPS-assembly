This program loops through a single precision floating point array of size 16, and fills up an int array of size 16 as such :
For every number in the input array,
	if it is bigger than the previous number  ->  write 1 in result array.
	if it is smaller than the previous number ->  write -1 in result array.	
	if it is equal to the previous number     ->  write 0 in result array.
